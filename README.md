# 🌐 ALEX // Personal Portfolio & Full-Stack Platform

Сучасний, швидкий та адаптивний веб-сайт портфоліо розробника з інтегрованою безсерверною архітектурою (Serverless Backend) для миттєвої обробки клієнтських запитів через Telegram API.

> **Унікальність архітектури:** Проєкт поєднує статичний високооптимізований фронтенд на GitHub Pages та асинхронний Python-бекенд (FastAPI), розгорнутий на платформі Vercel у вигляді Serverless Functions.

---

## 🚀 Навігація по сайту
* **Головна** — Інтродукція та позиціонування (Full-Stack Developer з архітектурним мисленням).
* **Стек** — Технологічний арсенал інструментів.
* **Проєкти** — Демонстрація реальних кейсів розробки (AlpsTravel, AlexiyATM).
* **Досвід** — Таймлайн професійного розвитку та шлях у розробці.
* **Контакти** — Скляна форма зворотного зв'язку з валідацією полів.

---

## 🛠️ Технологічний стек

### Backend (Serverless)
* **Python 3.10+** — Основна мова розробки серверної логіки.
* **FastAPI** — Асинхронний веб-фреймворк для побудови high-performance API ендпоінтів.
* **aiogram 3.x** — Сучасний асинхронний фреймворк для взаємодії з Telegram Bot API.
* **Pydantic v2** — Сувора валідація вхідних даних та JSON-схем.
* **Python-dotenv** — Безпечне керування секретами (`.env`) та змінними оточення.

### Frontend
* **JavaScript (ES6+)** — Асинхронна логіка відправки форм через `fetch` API, динамічні ефекти шапки сайту та бургер-меню.
* **HTML5 / CSS3** — Семантична структура та кастомні CSS-змінні.
* **Glassmorphism Design** — Сучасний UI-інтерфейс з ефектом розмиття (`backdrop-filter`) та неоновими акцентами.
* **Чуйна верстка** — Повна адаптивність під мобільні пристрої, планшети та монітори (Media Queries).

### DevOps
* **Vercel** — Хостинг для Serverless-ендпоінтів бекенду.
* **GitHub Pages** — Хостинг статичної частини клієнтського інтерфейсу.

## 📐 Архітектура обробки форми (Контакти)

1. Користувач заповнює форму (Ім'я, Email, Повідомлення) у секції контактів.
2. JavaScript перехоплює подію `submit`, блокує кнопку, перемикає текст на "Надсилання..." та формує JSON-об'єкт.
3. Клієнт робить асинхронний `POST` запит на захищений ендпоінт Vercel.
4. FastAPI приймає запит, автоматично валідує його через схему Pydantic (перевіряє коректність Email).
5. Через `aiogram` сервер формує Markdown-повідомлення та миттєво пересилає його адміністратору в Telegram.
6. Клієнт отримує статус `200 OK`, форма очищується, а користувач бачить успішне сповіщення без перезавантаження сторінки.
